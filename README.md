# Portfolio Website

A modern, professional portfolio website built with Next.js, TypeScript, and Tailwind CSS. Designed for job applications and client outreach to international markets (UK, Europe, Gulf).

## Features

- **Modern Stack**: Next.js 14 with TypeScript
- **Styling**: Tailwind CSS with custom color palette
- **Animations**: Framer Motion for smooth interactions
- **Typography**: Inter/Roboto for body text, Fira Code for code snippets
- **Responsive**: Fully responsive design for all devices

## Color Palette

- **Primary**: Deep Navy (#102a43) / Charcoal (#2d2d2d)
- **Accent**: Electric Blue (#00bcd4) / Teal (#009688)
- **Background**: Light Mode with dark accents (professional for business clients)

## Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

### Build for Production

```bash
npm run build
npm start
```
